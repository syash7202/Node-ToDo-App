# ToDo Node Application used for CICD

Project setup :

`sudo apt install nodejs -y`

`sudo apt install npm -y`

`npm install`

`node app.js`

or setup by Docker :

`docker build -t node-todo-app .`

`docker run -p 8000:8000 -d todo-node-container `
